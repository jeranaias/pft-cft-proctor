# PFT/CFT Proctor

Mobile-first PFT/CFT calculator and proctor worksheet for USMC fitness test monitoring.

## Features

- **PFT Calculator** - Calculate Physical Fitness Test scores with age/gender adjustments
  - Pull-ups or Push-ups (upper body)
  - Plank (core)
  - 3-Mile Run or 5k Row (cardio)
  - Altitude adjustment for 4,500ft+

- **CFT Calculator** - Calculate Combat Fitness Test scores
  - Movement to Contact (880 yards)
  - Ammunition Lift (30lb ammo can)
  - Maneuver Under Fire (300 yards)

- **Height/Weight & Body Fat** - BCP assessment tool
  - DoD height/weight standards
  - Tape test calculator
  - Body fat percentage calculation

- **Proctor Worksheet** - Track multiple Marines in a session
  - Enter Marine info and scores
  - Auto-calculate points and classification
  - Generate PDF reports

## Additional Features

- Night mode (red on black for field use)
- Offline capable (PWA)
- Works on mobile devices
- No account required

## Usage

Visit [https://jeranaias.github.io/pft-cft-proctor](https://jeranaias.github.io/pft-cft-proctor)

Or clone and run locally:

```bash
git clone https://github.com/jeranaias/pft-cft-proctor.git
cd pft-cft-proctor
# Open index.html in a browser
```
